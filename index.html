Ph235 Orbital Dynamics Sim

Final Project for Physics Simulations

View project on GitHub

Project Scope

The project will simulate a multi-stage rocket, launched from Earth into a sequence of increasingly complex orbits. The final orbit will deal with Chaos Theory, with the rocket orbiting a binary star.

Code Framework

The simulation will be built in Python, using VPython to display a visual representation of the rocket's trajectory. 4th Order Runge-Kutta will be used to solve the various equations necessary.

Authors and Contributors
